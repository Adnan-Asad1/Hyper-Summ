PROJECT OVERVIEW:
--------------------------------------------------------------------------------
This project implements an automated text summarization system using TF-IDF 
(Term Frequency-Inverse Document Frequency) technique with both sequential 
and parallel processing approaches.

DATASET:
--------------------------------------------------------------------------------
Dataset: CNN Daily News (Synthetic Sample)
Total Articles: 5 articles for testing
Text Processing: Paragraph-based chunking with dynamic summary length
Summary Length: 30-40% of original sentences (min 3, max 10 sentences)

IMPLEMENTATION DETAILS:
--------------------------------------------------------------------------------
1. Sequential Approach (Baseline):
   - Single-threaded TF-IDF computation
   - Processes entire document at once
   - Computes sentence importance scores
   - Extracts top sentences for summary

2. Parallel Approach (Multiprocessing):
   - Multi-process TF-IDF computation
   - Splits text into 1 chunks (avg)
   - Processes chunks in parallel across 4 CPU cores
   - Aggregates results from all processes
