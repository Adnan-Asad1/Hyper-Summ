## Summary Length Guarantee

The `calculate_dynamic_summary_length()` function ensures that:
- Summary is **always shorter** than the original text
- Summary length is 30-35% of original sentence count
- Minimum 1 sentence, maximum (original sentences - 1)
- This guarantees proper text compression
